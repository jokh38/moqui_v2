Detailed Description

template<typename T, typename R>
class mqi::aperture3d< T, R >

A 3D aperture model represented by a rectilinear grid.

This class extends grid3d to model a 3D aperture with one or more openings. It is a template class that can be used with different data types for grid values and coordinates. It is designed to be used in Monte Carlo particle transport simulations and supports CUDA for GPU acceleration.

Template Parameters
TThe data type of the grid values (e.g., dose, HU).
RThe data type of the grid coordinates (e.g., float, double).
