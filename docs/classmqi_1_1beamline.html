Detailed Description

template<typename T>
class mqi::beamline< T >

Represents a treatment machine beamline.

This class holds a collection of mqi::geometry objects that together define the physical structure of a treatment beamline. It allows for the dynamic addition of components and provides access to the entire collection.

Template Parameters
TThe data type for numerical values (e.g., float, double).
Note
The necessity of this class as more than a simple container is yet to be determined.
