Detailed Description

template<typename T>
class mqi::gtr2< T >

Represents the beam model for the Sumitomo IMPT machine (Gantry 2) at Samsung Medical Center (SMC).

This class provides a specific implementation of treatment_machine_ion for the GTR2 machine, primarily based on log file data. It uses spline interpolation to model various energy-dependent beam properties like spot size, angular spread, and particle count calibration.

Template Parameters
TThe floating-point type for phase-space variables (e.g., float or double).

Creates a beam source model from log file data.

This implementation is specific to the Samsung Medical Center's log file format. It iterates through the parsed log file data, characterizes each spot, and appends the resulting beamlets to the beam source.

Parameters

pcoordThe coordinate system of the beam geometry.
source_to_isocenter_mmThe distance from the source to the isocenter in mm.
rsuseA boolean indicating whether a range shifter is used.
Returns
A mqi::beamsource<T> object for the simulation.
