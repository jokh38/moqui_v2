Detailed Description

template<typename R>
class mqi::po_inelastic< R >

Represents the inelastic interaction between a proton and an oxygen nucleus.

This class calculates the cross-section for p-O inelastic scattering based on an analytical model. It serves as a base class for more complex models, such as those using tabulated data.

Template Parameters
RThe floating-point type (e.g., float, double).
