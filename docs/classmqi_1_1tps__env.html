Detailed Description

template<typename R>
class mqi::tps_env< R >

Manages a simulation environment based on a Treatment Planning System (TPS).

Template Parameters
RThe floating-point type used for simulation (e.g., float or double).

This class extends x_environment to create a simulation setup based on data from a TPS. It reads and parses DICOM files (CT, RTPLAN, RTSTRUCT) and log files to configure the patient geometry, materials, beam source, and scorers.
