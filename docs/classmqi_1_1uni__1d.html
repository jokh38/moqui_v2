Detailed Description

template<typename T>
class mqi::uni_1d< T >

A 1-dimensional uniform probability distribution function.

Template Parameters
TThe floating-point type of the distribution (e.g., float or double).

This class provides a 1D uniform distribution. It samples random values from a specified range [min, max].
