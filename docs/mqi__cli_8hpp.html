Detailed Description

Defines a command-line interface (CLI) helper class.

This file provides the cli class, which is designed to parse and manage command-line arguments for executables like unit tests and simulation applications. It allows a user to specify settings like input file paths, simulation parameters, and output options from the command line, making the programs flexible and configurable without needing to recompile the code.
