Detailed Description

Defines a collection of functions for saving simulation data to various file formats.

This file provides a set of input/output (I/O) utilities to export data from the simulation's internal structures (like scorers) into standard file formats that can be easily used by external analysis and visualization tools. Supported formats include raw binary, NumPy arrays (.npz), and medical imaging formats (.mhd, .mha).
