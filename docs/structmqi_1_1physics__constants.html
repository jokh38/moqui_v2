Detailed Description

template<typename R>
struct mqi::physics_constants< R >

A collection of fundamental physical constants and unit conversions.

Template Parameters
RThe floating-point type (e.g., float or double) used for the constants.

This struct is a container for constants used in physics calculations. By using a template, all constants can be defined with a specific numerical precision (float or double) at compile time. All members are const, making instances of this struct immutable, which is a safe practice for handling constants.

The base units used are millimeters (mm) for length and Mega-electron Volts (MeV) for energy.
