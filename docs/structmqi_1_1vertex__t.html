Detailed Description

template<typename T>
struct mqi::vertex_t< T >

Represents the state of a particle at a specific point in space and time (a vertex).

This structure holds the essential properties of a particle at a vertex, which is a point along its track. Various physics processes, geometry boundaries, or other simulation constraints can propose the next vertex for a particle.

Template Parameters
TThe numeric type for the vertex properties (e.g., float or double).
Note
This structure does not include the particle type itself, which is managed separately.
