Detailed Description

Main entry point for running a Monte Carlo simulation using the Treatment Planning System (TPS) environment.

This executable initializes the mqi::tps_env from an input file, runs the simulation, and reports the total execution time. It serves as a command-line interface for the Moqui simulation engine.
