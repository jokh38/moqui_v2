Detailed Description

template<typename T>
class mqi::const_1d< T >

A 1-dimensional constant probability distribution function (PDF).

This class inherits from pdf_Md and models a 1D distribution that always returns a constant value (the mean). The standard deviation parameter is ignored. It is useful for representing fixed-value parameters, such as a monoenergetic beam.

Template Parameters
TThe data type of the return value (e.g., float, double).
