Detailed Description

template<typename T, typename R>
class mqi::grid3d< T, R >

A template class for a 3D rectilinear grid.

This class is a fundamental data structure used to represent any 3D voxelized data, such as a patient's CT scan or a dose calculation grid.

Template Parameters
TThe data type of the value stored in each grid cell (e.g., int16_t for CT data, float for dose).
RThe floating-point type for the grid's spatial coordinates (e.g., float or double).
