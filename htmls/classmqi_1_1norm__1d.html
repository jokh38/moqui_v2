Detailed Description

template<typename T>
class mqi::norm_1d< T >

A 1-dimensional normal (Gaussian) probability distribution function (PDF).

This class inherits from pdf_Md and models a 1D Gaussian distribution. It uses a std::normal_distribution object to generate random numbers according to the specified mean and standard deviation.
