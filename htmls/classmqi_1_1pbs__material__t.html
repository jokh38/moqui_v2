Detailed Description

template<typename R>
class mqi::pbs_material_t< R >

A material definition class for Pencil Beam Scanning (PBS) simulations.

This class extends patient_material_t and can be used to define materials based on Hounsfield Units (HU).

Template Parameters
RThe floating-point type for material properties (e.g., float or double).
