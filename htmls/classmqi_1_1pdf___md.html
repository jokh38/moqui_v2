Detailed Description

template<typename T, std::size_t M>
class mqi::pdf_Md< T, M >

A pure virtual base class for M-dimensional probability distribution functions (PDFs).

This class serves as the foundation for all other distribution classes. It stores the mean and standard deviation for each dimension and defines the interface for sampling the distribution via the operator() method.

Template Parameters
TThe data type of the distributed values (e.g., float, double).
MThe number of dimensions of the distribution.
