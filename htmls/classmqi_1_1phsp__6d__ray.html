Detailed Description

template<typename T>
class mqi::phsp_6d_ray< T >

A 6-dimensional phase-space distribution for a ray-like source.

Template Parameters
TThe floating-point type of the distribution (e.g., float or double).

This class models a 6D Gaussian distribution for phase-space variables (x, y, z, x', y', z'). It is designed to simulate particle beams that can be approximated as rays, taking into account the correlations between position and direction (divergence).
