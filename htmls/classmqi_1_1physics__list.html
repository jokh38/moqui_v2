Detailed Description

template<typename R>
class mqi::physics_list< R >

A class to manage physics parameters and settings for the simulation.

Template Parameters
RThe floating-point type for energy cuts and step limits.

This class holds various energy cutoffs and step size limits that control the behavior of physics interactions.
