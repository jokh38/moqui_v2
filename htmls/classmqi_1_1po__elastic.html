Detailed Description

template<typename R>
class mqi::po_elastic< R >

An analytical model for proton-oxygen elastic scattering.

Template Parameters
RThe floating-point type for calculations.

This class implements an analytical cross-section model for p-O elastic scattering and handles the final state generation (energy transfer and scattering angle).
