Detailed Description

template<typename T>
class mqi::vec2< T >

A 2D vector class for position and direction representation.

This class provides basic vector operations for 2D coordinates. It is designed to be compatible with both CPU and CUDA device code.

Template Parameters
TThe numeric type of the vector elements (e.g., float or double).
