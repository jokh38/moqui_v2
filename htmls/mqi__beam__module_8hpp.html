Detailed Description

Defines the top-level abstraction for interpreting DICOM beam modules.

This file contains the declaration of the beam_module class, which serves as a base class for handling different types of DICOM beam modules, such as those for conventional radiotherapy (RT) and ion therapy (RT-Ion).
