Detailed Description

Defines mathematical constants and functions for both CPU and CUDA execution.

This header provides a hardware-abstraction layer for common mathematical operations like logarithms, square roots, and random number generation. It uses preprocessor directives (#if defined(__CUDACC__)) to select the appropriate implementation (e.g., standard C++ <cmath> functions or CUDA math.h functions) at compile time. This allows the same high-level simulation code to be compiled for and run on either the CPU or the GPU without modification.
