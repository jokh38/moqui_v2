Inserts a key-value pair into a hash table using atomic operations.

This function is designed for concurrent insertion into a shared hash table on the GPU. It uses linear probing to resolve hash collisions.

Template Parameters

CUDA kernel for transporting particles through a patient geometry.

This is the main particle transport loop. It processes a batch of particles (vertices), simulates their interaction with the patient geometry, and scores quantities like dose.

Template Parameters
RThe floating-point type (e.g., float, double).
Parameters

n_vtxThe total number of vertices to process.

scorer_offset_vectorAn optional vector for spot-based scoring.
score_local_depositA flag to enable/disable local energy deposition scoring.

CUDA kernel for transporting particles with per-particle seeding.

This version of the transport kernel is similar to transport_particles_patient but initializes the random number generator for each particle using a provided seed. This is useful for reproducibility and debugging.
