Detailed Description

Represents a single spot from a treatment log file for MC simulation.

This struct holds data for a spot as it was actually delivered, according to the machine's log files. This can be used for quality assurance or for simulations based on the delivered treatment rather than the planned one.

Note
Added in 2023 for SMC Log-file based MC simulation.
