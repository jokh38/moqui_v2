Detailed Description

template<typename R>
struct mqi::intersect_t< R >

Describes the result of a ray-tracing intersection with a grid voxel.

This structure is used to return the results of calculating the intersection of a particle's path (a ray) with the boundaries of a voxel.

Template Parameters
RThe floating-point type (e.g., float or double) for coordinate calculations.
