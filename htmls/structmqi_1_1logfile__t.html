Detailed Description

Represents data for a single field in a treatment log file.

A struct in C++ is a way to group several related variables into one place. It's similar to a simple Python class without methods or a dictionary-like object. This struct holds data from log files generated by the treatment machine, which record the actual delivered beam spots.
